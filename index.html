1. Основы
    • Структура базового документа HTML.
    • Элементы <html>, <head>, <body>.
    • Использование тегов, атрибутов и значений.

  2. Текстовые элементы
    • Заголовки <h1> - <h6>.
    • Параграфы <p>.
    • Отступы и переносы строк.
    • Выделение текста: <em>, <strong>, <mark>, <del> и др.

  3. Ссылки и медиа
    • Гиперссылки <a>.
    • Вставка изображений <img>.
    • Видео и аудио в HTML5.

  4. Списки
    • Нумерованные <ol> и маркированные <ul> списки.
    • Списки определений <dl>, <dt> и <dd>.

  5. Таблицы
    • Создание таблиц с помощью <table>, <tr>, <th>, <td> и других элементов.
    • Комбинирование ячеек и объединение строк и столбцов.

  6. Формы
    • Создание форм для ввода данных с помощью <form>, <input>, <textarea>, <select> и др.
    • Различные типы полей ввода, такие как текстовые поля, кнопки, флажки, переключатели и т.д.
    • Валидация форм.

  7. Блоки и контейнеры
    • Группировка элементов с помощью <div> и <span>.
    • Семантические блочные элементы: <header>, <nav>, <main>, <footer> и т.д.

  8. Метаданные и настройки
    • Метатеги <meta> для указания информации о документе (кодировка, описание, ключевые слова и т.д.).
    • Заголовки страницы <title>.
    • Комментарии <!-- -->.

  9. Семантика
    • Использование семантических тегов для повышения доступности и SEO (например, <article>, <section>, <nav>, <aside> и т.д.).


Основы HTML
  • <!DOCTYPE html> - одинарный тег для указание версии html
  • <html> - основной тег
  • <head>- тег для скрытой информации (настройки). Возможные теги для тега <head>:
    • <title> и </title> - при помощи этого тега можно описать какое название будет у конкретной определённой страницы
    • <meta> - одинарный тег для метаданных.
    • <link> - одинарный тег для указания отношения между текущим документом и связанным документом/ресурсом.
    • Возможные атрибуты для тега <link>:
      • rel - атрибут который определяет отношения между текущим документом и связанным документом/ресурсом.
      • Возможные значение которые может принимать атрибут rel:
        • stylesheet - значение для указания использования таблиц стилей
      • href - путь до файла
        • <script> - тег для подключения JavaScript
        • Возможные атрибуты для тега <script>:
        • src - путь до файла
  • <body> - тег в котором все теги будут видны конечному пользователю
  • <!-- Комментарий --> - комментарий
  • Порядок закрытия тегов - при использовании нескольких тегов для одного элемента. Нужно отделять теги и закрывать вложенный тег, а потом закрыть "родительский тег". Пример: <p>Просто <b>текст</b></p> Тут будет выглядеть как: Просто текст
  • Атрибуты применяемые ко всем тегам:
    • title - всплывающая подсказка при наведении на элемент
    • width - ширина элемента в пикселях
      • height - высота элемента в пикселях
Текстовые елементи
  • <h1> - <h6> - теги для заголовков (heading)
  • <p> - тег для параграфов (paragraph)
  • <b> - тег для жирного шрифта (bold)
  • <strong> - тег для выделения текста (делает его жирным)
  • <i> - тег для курсивного текста (italic)
  • <em> - тег для текста с акцентом или эмфазисом (делает его курсивным)
  • <strike> - тег для зачёркнутого текста
  • <sub> - subscript
  • <sup> - superscript
  • <br> - одинарный тег "заглушка". Используется для переноса строки (break row)
  • <hr> - одинарный тег для горизонтальной линии (horizontal row)
  • <abbr> - тег для аббревиатуры (abbreviation), с атрибутом title для расшифровки.
  • <address> - тег для указания адреса (делает его курсивным)
  • <big> - тег для увеличения текста
  • <small> - тег для уменьшения текста
  • <pre> - тег для предварительно отформатированного текста, сохраняющего пробелы и переносы строк (делает текст моноширинным)
  • <code> - тег для обозначения фрагментов кода. (делает текст моноширинным)
  • Возможные атрибуты для текстовых тегов:
    • title - текст появляющийся при наведении на текст внутри тега. (Часто используется для расшифровки аббревиатур)
Ссылки и медиа

  • Ссылки

    • <a> - тег для создания ссылки для текста
      • Возможные атрибуты для тега <a>:
        • name - атрибут для создания якорных ссылок, чтобы переходить к определенному элементу на странице
        • target - атрибут для указания метода открытия ссылки. Возможные значения атрибута target:
          • _self - значение по умолчанию, открывает ссылку на той же вкладке
          • _blank - открывает ссылку в новой вкладке
          • _parent - открывает ссылку в родительском фрейме, если такой есть.
        • href - атрибут для указания адреса, куда будет производиться переход при клике на ссылку. Возможные значения:
          • URL адрес
          • Путь к html файлу в текущей директории
          • #<Имя> - создание ссылки "якоря" при нажатии на которую выполнится перенос на выбранный элемент (используя атрибут name для ссылки).
          • mailto:<Почта> - перенаправление на почтовый клиент для написания письма на указанную почту

  • Медиа

    • <img> - одинарный тег для изображений (image). Используется для встраивания изображений на веб-страницу Возможные атрибуты для тега <img>:
      • src - атрибут для указания изображения (source). Возможные значения для атрибута src:
        • URL адрес изображения.
        • Путь к файлу изображения в текущей директории.
      • width и height - атрибуты для указания ширины и высоты изображения соответственно. Рекомендуется указывать оба атрибута для сохранения пропорций изображения. Возможные значения для атрибутов width и height:
        • auto - автоматическое указание значения
        • <Значение>% - для указания значения в процентах
        • <Значение>px - для указания значения в пикселях
      • alt - атрибут для указания текста, который будет отображен, если изображение не загрузилось, а также для улучшения доступности (alternative)
Списки
  • <ol> - тег для нумерованного списка (ordered list). Возможные атрибуты для тега <ol>:
    • start - для указания начала нумерации списка
    • type - для указания маркеров списка. В качестве маркеров могут быть следующие значения:
      • A - заглавные латинские буквы
      • a - строчные латинские буквы
      • I - заглавные римские цифры
      • i - строчные римские цифры
      • 1 - арабские цифры
    • reversed - для нумерации списка в обратном порядке
  • <ul> - тег для не нумерованного списка (unordered list). Возможные атрибуты для тега <ul>:
    • type - для указания маркеров списка. В качестве маркеров могут быть следующие значения:
      • disc - кружок
      • circle - окружность
      • square - квадрат
  • <li> - тег для указания элемента списка (list item)
Таблицы
  • <table> - тег для таблиц. Возможные вложенные теги для тега <table>:
    • <thead> - тег для указания шапки таблицы (table head).
    • <tfoot> - тег для указания футера таблицы (table foot).
    • <tbody> - тег для указания тела таблицы (table body). Возможные вложенные теги для тегов <thead>, <tfoot> и <tbody>:
    • <tr> - тег для указания ряда таблицы (table row). Возможные вложенные теги для тега <tr>:
    • <th> - тег для указания ячейки-заголовка таблицы (table header cell).
    • <td> - тег для указания элемента (ячейки) таблицы (table data cell). Возможные атрибуты для тегов <th> и <td>:

  • align - атрибут для указания горизонтального выравнивания текста в ячейке. Возможные значения для атрибута align:

    • left - выравнивание содержимого слева ячейки.
    • center - центрирование контента в ячейке.
    • right - выравнивание контента справа ячейки.

    • justify - вставляет пробелы в текстовое содержимое ячейки, чтобы содержимое заняло ячейку.

      Атрибут border и его использование лучше заменить на стилизацию с помощью CSS.

Формы
  • <form> - тег для создания формы
  • Возможные и обязательные атрибуты для тега <form>:
    • action - атрибут для указания страницы обработки данных формы
    • method - атрибут для указания HTTP метода передачи данных формы
  • Возможные вложенные теги для тега <form>:
    • <label> - тег для указания подписи нижнего поля
    • Возможные атрибуты для тега label:
      • for - атрибут для указания объекта с которым работает подпись внутри тега
    • <textarea> - тег для ввода текста (поддерживающий все атрибуты тега <input> за исключением атрибута type, value)
    • Возможные атрибуты для тега <textarea>:
      • cols - атрибут для указания ширины поля ввода
      • rows - атрибут для указания высоты поля ввода
    • <input> - одинарный тег для создания поля для ввода информации
    • Возможные атрибуты для тега <input>:
      • type - атрибут для указания типа поля (стоит учитывать что не все браузеры поддерживают все типы полей)
      • Возможные значения атрибута type:
        • text - для ввода текста
        • password -для ввода пароля (текст при вводе будет скрыт)
        • email - для ввода почты
        • url - для ввода url адреса
        • color - для выбора цвета
        • data - для ввода даты
        • time - для ввода времени
        • datetime - для ввода даты с временем
        • checkbox - для установки галочки
        • radio - для установки радио кнопки (кружочка) (для указания того что несколько радио полей (кружочков) используется для одного поля (ограничить максимально допустимое количество выборов), требуется указывать атрибут name)
        • file - для выбора файла
        • button - для создания кнопки
        • submit - для создания кнопки после нажатия на которую произойдёт отправка на сервер
        • reset - для очистки всех полей в форме
      • placeholder - атрибут для указания подсказки для поля
      • value - атрибут для установки (значения по умолчанию) какого либо текста внутри поля
      • maxlenght - атрибут для указания максимального количества символов для ввода
      • hidden - атрибут не принимающий какие либо значения для скрытия поля ввода
      • disabled - атрибут не принимающий какие либо значения для деактивации поля ввода
      • readonly - атрибут не принимающий какие либо значения для указания состояния поля ввода только для чтения
      • name - атрибут для указания имени поля ввода
      • required - атрибут для указания того что поля является обязательным
    • <select> - тег для создания выпадающего списка
    • Возможные атрибуты для тега <select>:
      • multiple - атрибут не принимающий какие либо значения для указания того что есть возможность сделать несколько выборов
      • size - атрибут для указания размера количества показываемых вариантов
    • Возможные вложенные теги для тега <select>:
      • <option> - тег для указания варианта выбора для выпадающего списка
      • Возможные атрибуты для тега <option>:
      • selected - атрибут не принимающий какие либо значения для указания значения по умолчанию
      • disabled - атрибут не принимающий какие либо значения для деактивации варианта
  • id - атрибут для указания уникального идентификатора ЛЮБОГО объекта
  • <button> - тег для создания кнопки
  • Возможные атрибуты для тега <button>
    • type - атрибут для указания типа кнопки
    • Возможные значения атрибута type:
      • button - для создания обычной кнопки
      • submit - для создания кнопки отправляющей данные из формы на сервер (для завершения формы)
      • reset - для создания кнопки очистки всех полей в форме
  • <progress> - тег прогресса (без указания атрибутов он может выступать в роли тега загрузки). Браузера не поддерживающие данный тег будут отображать содержимый текст
  • Возможные атрибуты для тега <progress>:
    • max - атрибут для указания максимального значения
    • value - атрибут для указания значения прогресса
Блоки и контейнеры

  • <div> - блочный тег, используемый для группировки элементов и создания блочных контейнеров на веб-странице. Он позволяет создавать отдельные секции и применять к ним стили через CSS.

  • <span> - строчный тег, используемый для выделения конкретных частей текста или вложения внутрь других элементов. Он также позволяет применять стили через CSS и является важным инструментом для изменения стиля или взаимодействия с текстом на странице.

Метаданные и настройки

Метаданные - это различные настройки которые можно установить к какой-то конкретной странице. Возможные атрибуты для одинарного тега <meta>:

  • charset - Установка кодировки
  • name - атрибут тега. Значение которого указывается в атрибуте content Возможные описания атрибута name:
    • author - имя автора
    • description - описание страницы
    • keywords - ключевые слова для страницы
  • content - для описания контента на странице
Семантика
  • <header> - блочный тег для шапки сайта
  • <main> - блочный тег для основной части сайта
  • <aside> - блочной тег для боковой части сайта
  • <footer> - блочный тег для футера сайта
  • <details> - тег для скрытого текста который будет показываться лишь при нажатии на выпадающий список
  • Вложенные теги для тега <details>:
    • <summary> - тег для создания кнопки для разкрития списка
